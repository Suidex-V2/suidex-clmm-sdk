# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this SDK or the SuiDex V3 CLMM protocol, please report it responsibly.

**Email:** security@suidex.org

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact assessment

We aim to acknowledge reports within 48 hours and provide a fix timeline within 5 business days.

## Scope

- `@suidex/clmm-sdk` TypeScript SDK
- SuiDex V3 CLMM Move contracts (package `0xb5f529c1dcda6580a61bf7ee9fbd524b50be62f11044d137c8202c8cbace9e56`)
